兩週後老師和我會再看一次：也是一樣的評分標準

當週(同學)以及兩週後(老師、助教)的評分分數各佔 50% 
    
    
1. 成功建立本課程的 repository，並且建立 `week_1/` (`course_1/`, `hw_1/`, `README.md`)
    * 0 / 0.5 / 1 分
2. `hw_1/ hw_1_basic.R` 完成
    * 0 / 0.5 / 1 分
3. `hw_1/hw_1_guess_number.R` 完成

4. 詳細的 `README.md` 解說。 ex.當週課堂、家裡的進度
    * 0 / 0.5 / 1 / 1.5 分

* `0.5`分: 完成部分要求。 
    ex. 程式碼部分可以執行，但是有bug。
* `1`分: 完成要求。
